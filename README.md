# 📚 FocusEd AI – Team C Capstone Project

This repository contains the full documentation and supporting materials for **FocusEd AI**, a software engineering capstone project developed by **Team C** at the University of Maryland Global Campus (UMGC).

FocusEd AI is a web-based educational platform that integrates with existing Learning Management Systems (LMS) and provides AI-assisted tools for teachers and students. The project includes requirements, design, implementation, testing, deployment, and user support documentation.

---

## 📄 Project Documentation

Each document below serves a specific purpose within the software development lifecycle. Refer to the filenames for the full details.

### 📝 Project Plan  
Outlines the project scope, goals, timeline, team responsibilities, risk management, communication plan, and cost estimates.  
📎 **File:** `Team C Project Plan.docx`

### 📘 Software Requirements Specification (SRS)  
Defines all functional and non-functional requirements for the FocusEd AI system, including use cases and system behaviors.  
📎 **File:** `Team C SRS.docx`

### 🏗️ Technical Design Document (TDD)  
Provides the architectural design, component diagrams, database schema, UI mockups, and major technical decisions made during development.  
📎 **File:** `Team C Technical Design Document.docx`

### 🧪 Software Test Plan  
Details the testing strategy, types of tests, environment setup, tools used, and testing schedule to ensure quality assurance.  
📎 **File:** `Team C Software Test Plan.docx`

### ✅ Test Report  
Summarizes the execution of all test cases mapped to the requirements, including test results, defects found, and overall assessment.  
📎 **File:** `Team C Test Report (1).docx`

### 👩‍💻 Programmer’s Guide  
Instructions for developers to understand, configure, and contribute to the FocusEd AI system. Includes tech stack, setup steps, and code structure.  
📎 **File:** `Team C Programmer’s Guide.docx`

### 🚀 Deployment and Operations Guide  
Explains how to deploy and maintain the system in a production environment. Includes hosting, configuration, and monitoring instructions.  
📎 **File:** `Team C Deployment and Operations Guide.docx`

### 🙋 User Guide  
Step-by-step walkthroughs for both teachers and students to use FocusEd AI’s features effectively, including assignment creation, AI feedback, and chat interactions.  
📎 **File:** `Team C User Guide (1).docx`
